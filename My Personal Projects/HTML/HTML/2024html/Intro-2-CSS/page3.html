To create a new page:
1.) create a new file and name it "name.html" and add the HTML starter code!
2.) Link the index.html and whatever other html pages you want by using this:
link href="index.html" rel="html" type="text/html"
3.) Make sure that your page is also linked on index.html (the main page)
